# Docker - Production Environment

- Create and run a container

  ```bash
  $ docker compose up -d
  ```
  Open [http://localhost:8080](http://localhost:8080) with your browser to see the
  result.

- To stop a container

  ```bash
  $ docker compose stop
  ```
- To remove a container

  ```bash
  $ docker compose down
  ```
